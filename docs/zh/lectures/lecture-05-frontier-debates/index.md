# 前沿思辨：语言、视觉与世界的边界

> "硅谷已经被LLM催眠了，完全被它催眠了。"
>
> -- 谢赛宁

这句话是一个技术判断，关于两种对"智能"的根本性定义之争的判断，这场争论正在决定未来十年 AI 研究的走向。

本讲不给定论。四场争论，每场都把最锋利的论点摆出来，问题留给你自己回答。

- **争论一、二**（见"语言是鸦片与 Bitter Lesson"）：语言是工具，还是"鸦片"？Bitter Lesson 到底说了什么，LLM 是遵循了还是违背了它？
- **争论三**（见"世界模型与 LLM 的分工与收敛"）：世界模型和 LLM 是竞争关系还是互补？WAM 和 JEPA 各自押注了什么，两条路会最终收敛吗？
- **争论四**（见"数据从哪里来"）："下载人类"的时代，物理世界数据从哪里来？
- **各路线赌注与收尾问题**（见"各路线核心赌注与收尾问题"）：六条架构路线的核心押注，CWM 案例，以及带着三个问题往下走。
