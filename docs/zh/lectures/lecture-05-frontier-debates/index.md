# 前沿思辨：语言、视觉与世界的边界

> "硅谷已经被LLM催眠了，完全被它催眠了。"
>
> —— 谢赛宁

这句话是一个技术判断——关于两种对"智能"的根本性定义之争的判断，这场争论正在决定未来十年 AI 研究的走向。

本讲不给定论。五场争论，每场都把最锋利的论点摆出来，问题留给你自己回答。

- **争论一**：语言是工具，还是"鸦片"？
- **争论二**：Bitter Lesson 到底说了什么？LLM 是遵循了还是违背了它？
- **争论三**：AGI 是目标，还是伪命题？
- **争论四**：世界模型和 LLM 是竞争关系还是互补？
- **争论五**："下载人类"的时代——数据从哪里来？
