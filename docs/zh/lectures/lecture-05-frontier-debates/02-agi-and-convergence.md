# 争论三四：AGI 是伪命题，以及两条路是否收敛

## 争论三：AGI 是目标，还是伪命题？

### LeCun 的质疑

Yann LeCun 在这个问题上有一个不太常被引用的论断，谢赛宁在访谈中转述了他的核心观点：

人有 200 万根视觉神经纤维，但真正能处理、能看到的东西趋近于零，受到意识和神经带宽的严格限制。你的视野中心之外，你几乎什么都看不见，只是你的大脑在"编造"一个完整的视觉场景的幻觉。

人的智能是一个**非常特定化的智能**，不是通用的。人类不能在低温下生存，不能精确计算三位数乘法，不能同时注意多个焦点。我们的智能是进化在特定生态位里优化出来的结果。

"通用智能"的定义本身就是人类中心主义的，我们把自己的智能当成了"通用"的标准，然后去问"机器能不能达到人类水平"。但如果人的智能本来就不通用，这个问题本身就问错了。

谢赛宁补充了一个视角：

> "智能演进是一个连续的过程，人不是独一无二的。"

他引用了 Sutton 的一句话（和 Bitter Lesson 同一位 Sutton）：

> "打造出一只松鼠的智能，才是真正难的问题。"

### 黑猩猩实验：动物的世界模型

为了说明"智能是连续谱"这个观点，谢赛宁提到了动物认知研究里的一个经典实验：

实验者当着黑猩猩的面，把食物放进两个盒子中的一个，然后把黑猩猩拉出去很长时间，再带回来。关键步骤：回来之前，黑猩猩看到实验人员正在吃其中一个盒子里的食物。

黑猩猩的反应：它径直走向**另一个**盒子打开，取出食物。

这意味着什么？黑猩猩在自己不在场的情况下，维护了一个对世界状态的内部模型：它记住了哪个盒子有食物，它理解"吃掉"意味着食物消失，它推断出另一个盒子还有食物。这是某种形式的心智推理（Theory of Mind），对世界状态的内部表征和推理。

动物有世界模型。智能不是人类的专利。"通用"可能从一开始就不是一个清晰的目标。

### 反驳：泛化能力是本质区别

AGI 的支持者会说，"通用"是个程度问题，不是非此即彼的。人类的跨域泛化能力是前所未有的：你可以学任何语言、玩任何游戏、做任何职业、适应任何文化。松鼠不能写代码，不能学微积分，不能组建社会制度。这种跨域泛化能力，才是人类智能的本质特征，也是 AGI 真正要追求的目标。

把"通用"定义成"松鼠水平"，是在故意降低标准。如果 AI 系统能做到人类级别的跨域泛化，管它叫不叫 AGI，那就是我们追求的东西。

### 留给你

如果连松鼠的智能都难以完整复制，我们有没有可能在真正理解"通用"之前就先做到了"通用"？

或者换个方向：你怎么定义"通用"？有没有一个不是人类中心主义的定义？

## 争论四：世界模型和 LLM 是竞争关系还是互补？

### 谢赛宁的分工逻辑

在这一点上，谢赛宁出人意料地温和。他没有说"LLM 是死路"，而是说：

> "没有 LLM，Vision 也没办法拓展到现在这种真正的多模态智能的大的范畴。"

他的 AMI Labs 不排斥 LLM，甚至明确感谢 LLM 带来的进步，因为 LLM 解决了语言理解和指令跟随这两个关键问题，让视觉系统有了"语言接口"，世界模型才能接受自然语言指令。

他的判断是：两者处理的是不同维度的信息，各有主场：

| 范式 | 主场 |
|------|------|
| LLM | 数字空间的推理、代码、文本生成、知识检索 |
| 世界模型 | 物理世界的预测、机器人控制、工业感知、具身智能 |
| WAM | 两者的统一：video + action 联合建模，理解物理并接受语言指令 |

这个"WAM"（World Action Model）的概念值得单独说明。它的核心思路是：把视频理解和动作预测联合训练。不只是"看懂世界"，而是"看懂世界并知道怎么在里面行动"。

如果你用一个矩阵来理解现有的世界模型，横轴是"吃不吃动作信号"，纵轴是"预测什么"：
- **Dreamer**：吃动作，预测潜在状态，狭义世界模型，主动决策者
- **WAM**：吃动作，同时理解语义，两者的统一

### 关键张力：最终会收敛吗？

这里有一个让双方都不舒服的问题：

如果世界模型最终都要接语言（接受自然语言指令），LLM 最终都要接视觉（处理图像和视频），两者会不会最终收敛成同一个东西？

GPT-4o 能看图、能听声音、能生成图像。V-JEPA 2 不生成像素，只预测语义表征，但也需要语言来描述任务。两条路正在从两个方向接近同一个点。

也许"LLM vs 世界模型"的争论，本质上是关于**核心设计哲学**的争论，而不是关于最终产品形态的争论：
- 你是从语言出发，往物理理解扩展？
- 还是从物理感知出发，往语言接口扩展？

出发点不同，可能导致系统架构、训练数据、评估指标上的根本差异，即使最终功能看起来相似。

### 留给你

V-JEPA 2 不生成像素，只预测语义表征。GPT-4o 能看图、能生成图。你觉得五年后这两条路还是分开的吗？

如果它们收敛了，是谁"赢"了，语言派还是世界模型派？还是说，这个问题本身就问错了？
