# 内部模拟：世界模型的历史与直觉

朋友从三米外扔来一个网球，你甚至还没看清球的旋转，手就已经移到了正确的位置。这不是反射，也不是运气——**你的大脑在那零点几秒里模拟了这颗球的飞行轨迹**，预测它会落在哪里，然后提前指挥肌肉行动。

世界模型就是这个机制的计算版本：大脑（或 AI）在内部维护的一张"世界运转规律"的草图，让行动者可以在真实出手之前先在脑子里把结果演一遍。

本讲分三个部分：

- **思想基石**：从 Craik（1943）到预测编码、内模原理，再到广义 vs 狭义世界模型的概念边界
- **四个时代**：RNN 萌芽 → Ha & Schmidhuber → Dreamer → JEPA 的演化脉络
- **为什么是现在**：视频生成、具身智能、自动驾驶三条线交汇的背景，以及本课程路线图
