# 第二讲：观测编码与潜在动力学

Dreamer 的核心问题可以拆成两个：**如何压缩感知**，以及**如何预测未来**。本讲分两个部分，各解决一个问题。

- **Part A · 观测编码**：为什么要压缩？VAE 的编码-解码结构，ELBO 损失的直觉，CNN 编码器的结构
- **Part B · 潜在动力学**：从最简单的 GRU 出发，经由 MDN-RNN 的不确定性建模，到达 RSSM 的确定性/随机性双路径设计

建议顺序读完再动手项目。Part A 是 P01 的直接前置，Part B 是 P02 的直接前置。
